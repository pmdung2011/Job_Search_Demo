# Job_Search

A simple Web Application where end-user can search and apply for jobs. Admin can edit/add new jobs.

## Getting Started
### Installing

```
git clone https://github.com/pmdung2011/job_search_demo.git
cd BE; npm install
cd FE; npm install
```

### Executing program

```
cd BE; npm start
cd FE; npm start
```
